## string 简介
string 是 C++ 风格字符串，是标准模板库中的==类==
>string 和 C语言中 char\* 的区别
>string 是一个类，string内部封装了char\*，是一个char*容器
>C语言的 char\* 是一个指针，指向的是一个不能被修改的常量

**头文件** `#include <string>`
